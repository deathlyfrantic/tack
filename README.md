## tack

A recreation of [Gary Bernhardt's
Selecta](https://github.com/garybernhardt/selecta) written in C.

## Usage

```
make
cat (ls *.txt | ./tack)
```

See [Selecta's
readme](https://github.com/garybernhardt/selecta/blob/master/README.md) for a
thorough explanation of general use.
